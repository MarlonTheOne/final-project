# final-project

### Elementos usados:
- 1 Maquina virtual
- Ubuntu Server 22.04
- Docker 20.10.21
- Docker Compose



Se clona el repositorio e ingresamos a la carpeta "final-project"

```bash
git clone https://github.com/MarlonTheOne/final-project.git
ls -l
cd final-project/
```
